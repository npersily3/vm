This function will rescue a page table entry that has been unmapped by the trimming thread, but the contents of the virtual page are still on the physical page.

If the page corresponding to the virtual address is on the modified list, this function just simply removes it from the middle. If the page is on the standby list, this function removes it from the middle of the list and sets the corresponding disk space free.
