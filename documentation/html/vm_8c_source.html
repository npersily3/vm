187 // Test our very complicated usermode virtual implementation.
188 //
189 // We will control the virtual and physical address space management
190 // ourselves with the only two exceptions being that we will :
191 //
192 // 1. Ask the operating system for the physical pages we'll use to
193 // form our pool.
194 //
195 // 2. Ask the operating system to connect one of our virtual addresses
196 // to one of our physical pages (from our pool).
197 //
198 // We would do both of those operations ourselves but the operating
199 // system (for security reasons) does not allow us to.

204 // mappings, freeing them, trimming them, writing them out to backing
